# NTT DATA Android Training Projects

Mobile development workshops for NTT DATA — projects created as the foundation for Kotlin Bootcamp courses, designed and taught by Daniel Richter, Senior Instructor.

---

## 📱 About

This repository contains Android training projects developed for internal workshops, Kotlin Bootcamp programs, and hands-on mobile engineering sessions at NTT DATA.

The goal of these projects is to provide practical experience with modern Android development using Kotlin and current Android architecture guidelines.

---

## 🚀 Topics Covered

- Kotlin for Android
- Android SDK fundamentals
- Activity & Fragment lifecycle
- Jetpack Components
- REST API integration
- Coroutines
- MVVM Architecture
- Material Design
- Navigation Component
- Dependency Injection
- Firebase integration
- Location Services
- Local persistence
- Clean Architecture principles

## 🛠 Tech Stack

- Room Database
